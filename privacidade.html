Política de Privacidade

Última atualização: 14 de maio de 2026

A Loja LL Samples vende produtos digitais, como samples, loops e materiais musicais. Esta política explica quais dados podemos coletar, como usamos esses dados e como o cliente pode entrar em contato.

1. Dados coletados

Durante uma compra ou contato, podemos coletar:

  • nome;
  • e-mail;
  • telefone ou WhatsApp;
  • itens comprados;
  • status do pagamento;
  • código de acesso criado para consultar compras;
  • informações técnicas básicas, como data/hora de acesso e página visitada.

2. Finalidade do uso dos dados

Usamos os dados para processar pedidos, confirmar pagamentos, liberar downloads, enviar e-mails de compra aprovada, prestar suporte e manter histórico de compras.

3. Pagamentos

Os pagamentos são processados por plataformas externas, como Mercado Pago e PayPal. A Loja LL Samples não armazena dados completos de cartão de crédito.

4. Entrega por e-mail e registro de pedidos

Podemos usar serviços como Resend para envio de e-mails de entrega, Vercel para hospedagem e GitHub para registro operacional dos pedidos e contatos necessários ao atendimento.

5. Cookies e armazenamento local

O site pode usar armazenamento local do navegador para melhorar a experiência, como carrinho, preferências e histórico local temporário. O cliente pode limpar esses dados nas configurações do navegador.

6. Compartilhamento de dados

Não vendemos dados pessoais. Dados podem ser compartilhados apenas com fornecedores necessários para funcionamento da loja, pagamento, hospedagem, envio de e-mail e suporte.

7. Direitos do cliente

O cliente pode solicitar consulta, correção ou exclusão de dados de contato e histórico operacional, quando aplicável, pelo WhatsApp ou e-mail informado nesta página.

8. Segurança

O site usa HTTPS e ferramentas de terceiros para processamento seguro de pagamentos. Apesar disso, nenhum sistema é totalmente imune a riscos, e adotamos medidas proporcionais para proteger as informações necessárias ao funcionamento da loja.
